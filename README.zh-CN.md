# AI 短剧世界观设定工具包

短剧世界观设定，是在剧本、角色、场景和视频生成之前，为故事宇宙建立规则。本仓库整理故事 bible、地点系统、人物关系和连续性记录模板，适合 AI 短剧和竖屏微短剧生产。

它适合希望在进入 script-to-video 或分镜工作流前，先建立可复用故事基础的创作者。

## 为什么世界观重要

短剧生产里，世界观薄弱会导致：

- 角色决策前后不一致。
- 地点变化没有剧情原因。
- 社会规则和权力结构不清楚。
- 跨集冲突重复。
- AI 生成场景时忘记前文事实。

世界观工具包的作用，是为团队建立一个共同的故事事实源。

## 世界观层级

| 层级 | 定义什么 | 为什么重要 |
|---|---|---|
| 前提 | 核心故事承诺和类型 | 保持系列聚焦 |
| 规则 | 社会、情绪或超自然规则 | 避免剧情逻辑随机 |
| 地点 | 关键地点和视觉锚点 | 帮助分镜和场景生成 |
| 权力关系 | 谁对谁有影响力 | 驱动戏剧冲突 |
| 时间线 | 过去事件和集数顺序 | 支持连续性 |
| 秘密 | 隐藏信息和揭示顺序 | 控制悬念 |

## 生产流程

1. 定义前提和类型承诺。
2. 建立世界规则和冲突引擎。
3. 创建常用地点和视觉锚点。
4. 绘制人物权力关系。
5. 追踪秘密、揭示和跨集记忆。
6. 在写剧本或生成分镜前使用世界观 bible。
