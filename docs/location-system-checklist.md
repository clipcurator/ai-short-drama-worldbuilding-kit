# Location System Checklist

Locations in short drama should be reusable, visually recognizable, and connected to story conflict.

## Reuse rules

- Does the location create immediate context?
- Does it help the audience understand power relationships?
- Can it be reused across episodes without feeling repetitive?
- Does it have a visual anchor for storyboard generation?
- Does the location change when the story changes?

## Common location types

| Type | Drama use |
|---|---|
| Home | Intimacy, family pressure, hidden secrets |
| Office | Power, competition, status conflict |
| Hospital | Urgency, vulnerability, life-changing reveal |
| Wedding venue | Public stakes, identity reveal, betrayal |
| Alley or street | Chase, danger, private confrontation |
